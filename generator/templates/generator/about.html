Password Generator

About the Project

This is my first project that uses Django! I started off by getting practice with urls.py and views.py, and understanding how it is all connected to the templates.

The goal was to create a Password generator that allows the user to choose between including uppercase letters, numbers and special characters in it. It uses the random Python library to choose random characters from a list of each type of possible character.
